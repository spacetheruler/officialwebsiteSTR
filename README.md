# SPACE THE RULER — Digital Headquarters

**spacetheruler.com**

The official digital universe of SPACE THE RULER — rapper, producer, and creative entrepreneur.

---

## What This Is

A self-contained artist platform built as a single deployable HTML file. No build step required. No dependencies to install. Drops straight onto Vercel.

---

## Deployment

This site is deployed via **GitHub + Vercel**.

Any push to the `main` branch automatically deploys to production.

---

## Project Structure

```
/
├── index.html       ← The entire platform (self-contained)
├── og-image.jpg     ← Social sharing preview image
├── favicon.png      ← Browser tab icon (512x512 source)
└── README.md        ← This file
```

---

## Tech Stack

- Pure HTML / CSS / JavaScript
- No frameworks, no build tools, no npm
- Google Fonts (Orbitron, Cormorant Garamond, Space Grotesk)
- Vercel static hosting
